# NewsCycle
NewsCycle is a webdev project that extracts data from news sites and displays it visually.
<img src="https://i.imgur.com/Kr17PDK.png" width="400"/>
## Project Structure
Angular is used for the frontend. Django is used for the backend. A sqlite database is used.
